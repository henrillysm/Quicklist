🛒 Quicklist
Aplicação para gerenciar a lista de compras da semana, desenvolvida como desafio do primeiro módulo da formação.

🎯 Sobre o Projeto
O Quicklist permite ao usuário adicionar, marcar como concluídos e remover itens de uma lista de compras. A interface é responsiva e conta com feedbacks visuais para as ações realizadas.

✨ Funcionalidades

Adicionar novos itens à lista
Marcar itens como concluídos
Remover itens da lista
Feedback visual ao remover um item
Validação de campo vazio ao adicionar
Layout responsivo para mobile e desktop


🛠️ Tecnologias

HTML5
CSS3
JavaScript
